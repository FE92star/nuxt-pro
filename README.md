# nuxt-pro

> 一个nux项目用于熟悉vue ssr开发流程

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
# 本地开发模式
$ yarn run dev

# build for production and launch server
# 生产环境构建并且运行再服务端
$ yarn run build
$ yarn start

# generate static project
# 打包成静态资源用于部署静态网站
$ yarn run generate
